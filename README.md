Project 2 - HTML Foundation Page 
- The theme of my webpage is a celebration of the unappreciated English rock band Cardiacs.
- Creating this page was a great lesson (refresher) of using HTML to make a simple webpage. I have done some HTML webpage creation MANY years ago, just for fun, and had forgotton so much. This was a great chance to brush back up on my HTML and how to structure it properly (syntax, elements, ect..)

My code passed the validation completely before I added the style elements (center, font color, bg color) and now the only "errors" I have through the validator are "Use CSS instead", which is the next project so I'm leaving as is.
